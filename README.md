# 🚀 Portfólio Digital | Lucas Santana

Este projeto é a minha vitrine profissional, desenvolvida para o desafio da **IT Trends** (Fase 1). Ele foi construído com foco em **performance extrema**, **acessibilidade** e **semântica**, unindo as áreas de Desenvolvimento Web e Análise de Dados.

🌐 **Visualize o projeto online:** https://portfolio-interativo-six.vercel.app/

## 🛠️ Decisões de Arquitetura

Para garantir que este projeto saisse do básico, apliquei conceitos fundamentais de engenharia de software:

* **HTML5 Semântico:** Uso de tags estruturais para garantir que o site seja legível tanto por humanos quanto por algoritmos de busca (SEO).
* **CSS3 & Flexbox:** Implementação de um layout **Mobile-First**, garantindo que a experiência no telemóvel seja a prioridade, sem perder a elegibilidade no desktop.
* **Identidade Cyber-Data:** Escolha de uma paleta de cores neon sobre fundo escuro para refletir minha atuação com Python e agentes de IA.
